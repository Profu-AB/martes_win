Installationsinstruktioner för Martes

Hur installeras Martes?

  • Martes är en klassisk webbapplikation som kommer att köras i din webbläsare.
  • För att kunna installera detta på din dator kommer vi att använda oss Windows Subsystem for Linux eller WSL.
  • Om WSL inte finns på din dator sedan tidigare behöver vi installera det först.
    • Denna installation kommer då att kräva att du kan logga in med administratörsrättigheter på din dator.
  • Installationsprocessen för Martes kommer att kontrollera om WSL finns sedan tidigare och installera det om det inte gör det.

Kontrollera systemkraven

  • Operativsystem: Windows 10 (build 19041 eller senare) eller Windows 11
  • Ledigt diskutrymme: minst 4GB utrymme kommer att krävs.
  • Behörigheter: Temporär tillgång till administratörsrättigheter kan krävas

Tips: Så här kontrollerar du din Windows Version:

  • Tryck på Win + R för att öppna dialogrutan Kör.
  • Skriv winver och tryck på Enter.
  • Kör du Windows 10 kontrollera att att OS Build är minst 19041.

Licenskod

  1. Profu kommer att skicka en licenskod till dig.
  2. Denna kod måste du använda på två ställen
    1. Kör bat filen setup_env.bat och skicka med din kod som parameter
      • Exempel setup_env.bat MINLICENSKOD
    2. När du startat Martes i webläsaren för första gången klickar du på ikonen med ett "?" uppe i högra hörnet
      • Under rubriken Licensnyckel finns ett textfält. Här klistrar du in din licenskod och trycker på knappen "Aktivera"

Installation

  1. Öppna programmet Windows Terminal eller en kommandotolken (cmd.exe) med administratörsrättigheter genom att:
    • Sök efter "Terminal" eller "cmd" i Start-menyn.
    • Högerklicka på "Windows Terminal" eller "Kommandtolken" och välj Kör som administratör.
  2. se till att du står i mappen där du placerade martes installationsfiler och skriv in följande :
      3. cd install
      install.bat
  3. Om instruktiner kommer om att Du behöver starta om din dator så gör du det.
    • Efter återstart går du återigen till installations-mappen och kör samma script install.bat igen.
  4. Nu är installationen klar och din webbläsare bör ha öppnats med http://localhost:8080

Avinstallation

  • För att avinstallera Martes går du till mappen install och kör uninstall.bat
