Installationsinstruktioner

1. Hur installeras Martes?

Martes är en webbapplikation som körs lokalt i din webbläsare. Under huven använder den Docker-containrar som körs i Windows Subsystem for Linux (WSL).

Installationsprocessen kontrollerar automatiskt om WSL finns sedan tidigare och installerar det om det behövs.

2. Systemkrav

OperativsystemWindows 10 (build 19041+) eller Windows 11
DiskutrymmeMinst 4 GB ledigt
BehörigheterAdministratörsrättigheter krävs vid första installationen

Tips: Kontrollera din Windows-version genom att trycka Win + R, skriva winver och trycka Enter.

3. Installation

  1. Öppna Windows Terminal eller Kommandotolken (cmd) som administratör:
    • Sök efter "Terminal" eller "cmd" i Start-menyn
    • Högerklicka och välj Kör som administratör
  2. Navigera till mappen där du placerade Martes installationsfiler: 
    cd install
install.bat
  3. Om du uppmanas att starta om datorn: gör det, gå sedan tillbaka till mappen och kör install.bat igen.
  4. Installationen är klar! Din webbläsare öppnas automatiskt med http://localhost:8080

4. Licenskod

Du får en licenskod från Profu. Den behöver anges på två ställen:

  1. I kommandotolken, kör: 
    setup_env.bat DIN_LICENSKOD
  2. I Martes webbgränssnitt:
    • Klicka på ?-ikonen uppe till höger
    • Under Licensnyckel, klistra in din kod och klicka Aktivera

5. Uppdatering

För att uppdatera Martes till senaste versionen:

cd install
update.bat

Scriptet laddar ner de senaste versionerna och startar om tjänsterna. Dina data och inställningar bevaras.

Obs: Om du har Joule AI med Ollama installerat uppdateras den separat:

cd install
update_ollama.bat

6. Joule AI (valfritt)

Först: Kör gitpull.bat i martes_win-mappen för att hämta de senaste installationsfilerna.

Joule AI är Martes inbyggda AI-assistent. Den kan svara på frågor om dina indatauppsättningar, köra simuleringar och generera rapporter.

Det finns två sätt att använda Joule AI:

Alternativ A: Anthropic Claude (molnbaserad)

Kräver internetåtkomst och en API-nyckel från Profu.

  1. Öppna filen install\.env i en texteditor
  2. Lägg till följande rader: 
    JOULE_PROVIDER=anthropic
ANTHROPIC_API_KEY=din-api-nyckel-här
  3. Starta om Martes: 
    cd install
update.bat

Alternativ B: Ollama (lokal, gratis)

Kör AI-modellen lokalt på din dator. Kräver ~11 GB extra diskutrymme. En NVIDIA GPU rekommenderas för snabba svar.

  1. Se till att Martes är startad (kör start.bat om det behövs)
  2. Installera Ollama: 
    cd install
install_ollama.bat

    Nedladdningen är ca 11 GB och kan ta en stund.

  3. Öppna filen install\.env och lägg till: 
    JOULE_PROVIDER=ollama
  4. Starta om Martes: 
    cd install
update.bat

Uppdatera Ollama

cd install
update_ollama.bat

Avinstallera Ollama

cd install
uninstall_ollama.bat

Detta tar bort Ollama-containern, modellen och frigör ~11 GB diskutrymme. Resten av Martes påverkas inte.

7. Avinstallation

För att avinstallera Martes helt:

cd install
uninstall.bat
