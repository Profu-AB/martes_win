Installationsinstruktioner för Martes

1. Kontrollera systemkraven

  • Operativsystem: Windows 10 (build 19041 eller senare) eller Windows 11
  • Behörigheter: Temporär tillgång till administratörsrättigheter kan krävas
  • Förberedelser:
    • WSL 2.0 (Windows Subsystem for Linux 2.0)
      • Martes består av 3 st sk Docker-images, vilka kommer att hanteras och köras i WSL (linux).
    • Ubuntu 22.04.05 LTS eller senare (från Microsoft Store)
      • Ubuntu är en populär Linux-distribution som, när den används med WSL, möjliggör en fullfjädrad Linux-miljö direkt i Windows.
    • Git för Windows (versionshantering)
      • Git används för att ladda ner de filer du behöver för att installera och starta Martes på din dator.
    • En webläsare av senaste version. Vi har testat med Chrome och Edge.

Tips: Så här kontrollerar du din Windows Version:

  • Tryck på Win + R för att öppna dialogrutan Kör.
  • Skriv winver och tryck på Enter.
  • Kör du Windows 10 kontrollera att att OS Build är minst 19041.

2. Installera WSL 2.0

Notera: Detta steg kräver administratörsrättigheter för din Windows användare.

  1. Öppna programmet Windows Terminal eller en kommandotolken (cmd.exe) med administratörsrättigheter genom att:
    • Sök efter "Terminal" eller "cmd" i Start-menyn.
    • Högerklicka på "Windows Terminal" eller "Kommandtolken" och välj Kör som administratör.
  2. Kör följande kommando för att installera WSL och ställa in version 2 som standard:
    • se till att du står i mappen där du placerade martes installationsfiler och skriv in följande :
      • cd install
      install_wsl.bat
    • Följ instruktionerna på skärmen och ange ett användarnamn och ett lösenord för din linux-användare.
      • Kom ihåg detta lösenord. Det kan komma att efterfrågas vid ett flertal tillfällen i efterföljande steg.
    • När detta är klart skriver du exit och trycker på retur för att komma tillbaka till kommandotolken i windows.
    • Du bör se text på slärmen ungefär som nedan exempel: 
      The operation completed successfully.

"Setting Ubuntu-22.04 as the default WSL distribution..."
The operation completed successfully.

"Ubuntu-22.04 is now being set up. Please complete the prompts to create a Linux username and password."

To run a command as administrator (user "root"), use "sudo ".
See "man sudo_root" for details.

"WSL setup is complete. You are now back at the Windows command prompt."

    NAME            STATE           VERSION
* Ubuntu-22.04    Stopped         2
    Ubuntu          Stopped         2
  3. Om detta var första gången WSL installerades på din dator så behöver du nu starta om Windows.
    • Efter återstart fortsätter du med nästa steg
  4. Nu är din Windows dator förbered för installationen av Martes.

4. Installera Martes på Windows

  1. Öppna kommandotolken eller Windows Terminal (sök efter "Terminal" i Start-menyn).
  2. Navigera till mappen martes_win: 
    cd martes_win
  3. Gå till installationsmappen: 
    cd install
  4. Kör installationsfilen: 
    install.bat
    • OBS! Om ditt terminal-förster inleder prompten med PS betyder det att du har öppnat en PopwerShell-terminal. Då måste du skriva &.\ 
        &.\install.bat
  5. Följ instruktionerna under körningen av install.bat för att slutföra installationen.
    • Observera att en del av stegen som genomförs kan ta en liten stund. Avbryt helst inte installationen utan låt den gå klart!
  6. Kontrollera installationen
    • Öppna din webbläsare och navigera till http://localhost:3000. Om Martes visas korrekt är installationen klar.
    • Du kan också testa adressen http://localhost:8000. Detta är simuleringsdelen av Martes. Det bör visas en sida här. Klickar du på texten Martes så slå gärna på ljudet på din dator först!
    • Om du har Mongodb Compass installerat i Windows kan du prova att öppna Martes databasen genom att ansluta till port 27018
    • Om dessa länkar inte fungerar prova att köra filen \start\start.bat för att starta upp Martes i WSL.
