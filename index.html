Installationsinstruktioner för Martes

1. Kontrollera systemkraven

  • Operativsystem: Windows 10 (build 19041 eller senare) eller Windows 11
  • Behörigheter: Temporär tillgång till administratörsrättigheter kan krävas
  • Förberedelser:
    • WSL 2.0 (Windows Subsystem for Linux 2.0)
      • Martes består av 3 st sk Docker-images, vilka kommer att hanteras och köras i WSL (linux).
    • Ubuntu 22.04.05 LTS eller senare (från Microsoft Store)
      • Ubuntu är en populär Linux-distribution som, när den används med WSL, möjliggör en fullfjädrad Linux-miljö direkt i Windows.
    • Git för Windows (versionshantering)
      • Git används för att ladda ner de filer du behöver för att installera och starta Martes på din dator.
    • En webläsare av senaste version. Vi har testat med Chrome och Edge.

Tips: Så här kontrollerar du din Windows Version:

  • Tryck på Win + R för att öppna dialogrutan Kör.
  • Skriv winver och tryck på Enter.
  • Kör du Windows 10 kontrollera att att OS Build är minst 19041.

2. Installera WSL 2.0

Notera: Om du redan har WSL 2.0 installerat kan du hoppa över detta steg. Detta steg kräver administratörsrättigheter.

  1. Öppna programmet Windows Terminal med administratörsrättigheter genom att:
    • Söka efter "Terminal" i Start-menyn.
    • Högerklicka på "Windows Terminal" och välja Kör som administratör.
  2. Kör följande kommando för att installera WSL och ställa in version 2 som standard: 
    wsl --install
  3. Bekräfta att WSL är korrekt installerat och använder version 2: 
    wsl --set-default-version 2

3. Installera och verifiera Ubuntu 22.04.05 LTS (eller senare)

Notera: Om du redan har Ubuntu 22.04.05 LTS installerat kan du hoppa över detta steg.

  1. Öppna Microsoft Store.
  2. Sök efter Ubuntu 22.04.05 LTS och klicka på Installera.
  3. När installationen är klar, öppna ett kommandofönster och kör: 
    wsl --list --verbose
    Du bör se något i stil med: 
    NAME            STATE           VERSION
* Ubuntu-22.04  Running         2
  4. Starta Ubuntu genom att skriva: 
    wsl
  5. Följ instruktionerna för att skapa ett användarnamn och lösenord för Linux-distributionen.

4. Installera Git i Windows

Notera: Om du redan har Git installerat kan du hoppa över detta steg.

  1. Ladda ner Git från Git for Windows ( 64-bit version of Git for Windows.).
    • Alternativt kan du från kommando-prompten köra 
      winget install --id Git.Git -e --source winget
      . (kräver administratörsrättigheter)
    • Om du använder winget kommer senaste version av git att laddas ner och installationsprogrammet kommer startas automastiskt.
    • Tips: Finns inte winget på din dator kan du installera App Installer från Microsoft Store här.
  2. Kör installationsprogrammet och följ anvisningarna (välj standardinställningar om du är osäker).
  3. Kontrollera installationen: 
    git --version
    Om en version visas (t.ex. git version 2.x.x) är Git korrekt installerat.

5. Installera Martes på Windows

  1. Öppna programmet Windows Terminal (sök efter "Terminal" i Start-menyn).
  2. Gå till en mapp där du vill installera Martes, t.ex. C:\: 
    cd C:\
  3. Kör följande kommando för att klona Martes-repositoriet: 
    git clone https://github.com/mmagnemyr/martes_win.git
  4. Navigera till mappen martes_win: 
    cd martes_win
  5. Gå till installationsmappen: 
    cd install
  6. Kör installationsfilen: 
    install.bat
    • OBS! Om ditt terminal förster inleder prompten med PS betyder det att du har öppnat en PopwerShell-terminal. Då måste du skriva &.\ 
        &.\install.bat
  7. Följ instruktionerna under körningen av install.bat för att slutföra installationen.
  8. Kontrollera installationen
    • Öppna din webbläsare och navigera till http://localhost:3000. Om Martes visas korrekt är installationen klar.
    • Du kan också testa adressen http://localhost:8000. Detta är simuleringsdelen av Martes. Det bör visas en sida här. Klickar du på texten Martes så slå gärna på ljudet på din dator först!
    • Om du har Mongodb Compass installerat i Windows kan du prova att öppna Martes databasen genom att ansluta till port 27018
    • Om dessa länkar inte fungerar prova att köra filen \start\start.bat för att starta upp Martes i WSL.
