Installationsinstruktioner för Martes

1. Kontrollera systemkraven

  • Operativsystem: Windows 10 eller Windows 11
  • Behörigheter: Temporär tillgång till administratörsrättigheter kan krävas
  • Förberedelser:
    • WSL 2.0 (Windows Subsystem for Linux 2.0)
    • Ubuntu 22.04.05 LTS eller senare (från Microsoft Store)
    • Git för Windows

2. Installera WSL 2.0

  1. Öppna ett kommandofönster (Command Prompt) med administratörsrättigheter.
  2. Kör följande kommando för att installera WSL och ställa in version 2 som standard: 
    wsl --install
  3. Bekräfta att WSL är korrekt installerat och använder version 2: 
    wsl --set-default-version 2

3. Installera och verifiera Ubuntu 22.04.05 LTS

  1. Öppna Microsoft Store.
  2. Sök efter Ubuntu 22.04.05 LTS och klicka på Installera.
  3. När installationen är klar, öppna ett kommandofönster och kör: 
    wsl --list --verbose
    Du bör se något i stil med: 
    NAME            STATE           VERSION
* Ubuntu-22.04  Running         2
  4. Starta Ubuntu genom att skriva: 
    wsl
  5. Följ instruktionerna för att skapa ett användarnamn och lösenord för Linux-distributionen.

4. Installera Git i Windows

  1. Ladda ner Git från Git for Windows.
  2. Kör installationsprogrammet och följ anvisningarna (välj standardinställningar om du är osäker).
  3. Kontrollera installationen: 
    git --version
    Om en version visas (t.ex. git version 2.x.x) är Git korrekt installerat.

5. Installera Martes på Windows

  1. Öppna ett kommandofönster (Command Prompt) med administratörsrättigheter.
  2. Gå till en mapp där du vill installera Martes, t.ex. C:\: 
    cd C:\
  3. Kör följande kommando för att klona Martes-repositoriet: 
    git clone https://github.com/mmagnemyr/martes_win.git
  4. Navigera in i den klonade mappen: 
    cd martes_win
  5. Gå till installationsmappen: 
    cd install
  6. Kör installationsfilen: 
    install.bat
  7. Följ instruktionerna under körningen av install.bat för att slutföra installationen.
